# Vendor KB Baseline — Research Notes (Golden Set, 12 Top Vendors)

**Purpose.** This is a *human-verified* baseline ("golden set") for 12 of Greenway's
top vendors. It exists so we can score the web crawler's accuracy: when the crawler
researches these same vendors, its output is compared field-by-field against the
facts recorded here. A field the crawler gets right that matches this doc is a
true positive; a field it invents or gets wrong is caught immediately.

**Strict no-guessing rule.** Every fact below is taken from a named source and
carries a confidence score. Where a vendor could not be confirmed from a reputable,
consistent source, it is marked **UNCONFIRMED** and **NOTHING is invented** — we
record the gap and flag it for the owner. We never fill a field from assumption.

**Sources used (in priority order):**
1. The vendor's own official website (highest trust for self-described facts:
   mission, brands, contact, history).
2. WA state records — L&I business lookup (`secure.lni.wa.gov`), WA LCB licensee
   data, and licensee-data aggregators that mirror LCB traceability
   (Top Shelf Data `topshelfdata.com`, yourweeddata `yourweeddata.com`).
3. Reputable cannabis directories (Leafly, AskGrowers) — used only as secondary
   corroboration, never as a sole source for hard facts like address/UBI.

**Confidence scale:** 0.90–0.95 = multiple consistent sources incl. official site
or state record; 0.75–0.89 = single strong source; 0.50–0.74 = single secondary
source only; below 0.50 = unconfirmed (NOT seeded as fact).

**Data quirk to remember:** WA "tradename" in LCB filings is often the legal entity
name, not the consumer brand. The public "Cannabis Applicants" list LCB publishes is
**retailers only** — producers/processors (which most of these vendors are) are not
in it, so we verified those via L&I + traceability aggregators instead.
